## Visão

Meu trabalho fica na interseção entre engenharia de software, IA aplicada e automação operacional.
Tenho interesse especial por sistemas úteis e auditáveis: publicação baseada em Git, tooling para agentes,
análise documental, OCR, workflows investigativos e aplicações de IA para o setor público.

Não me interessa “demo de tecnologia”. O foco é construir fluxos que funcionem sob restrições reais:
contexto incompleto, necessidade de validação, histórico de mudanças e operação contínua.

## Em Que Estou Trabalhando

- Tooling e workflows agentic para desenvolvimento assistido por IA
- Integrações entre LLMs, CLIs, MCPs e automação operacional
- Publicação de conteúdo e sites com GitHub, Markdown e Vercel
- Estruturas de conhecimento e operação com Obsidian
- Triagem investigativa, OCR e fluxos forenses com apoio de IA
- Sistemas voltados a produtividade, consistência e validação de contexto

## Projetos e Frentes

### JBA Desenvolvimento

Consultoria e operação de conteúdo em IA para o setor público, com fluxo de publicação baseado em repositório,
Markdown, GitHub e deploy automático na Vercel.

### Digital Garden / joao-filho

Base pública de conhecimento construída com notas em Markdown, GitHub e Vercel, usada como sistema de publicação,
autoridade e reaproveitamento de conteúdo.

### PiauEye

Arquitetura de plataforma de análise forense digital com IA híbrida, automação de ingestão, extração de entidades
e organização de evidências para contexto policial brasileiro.

### Agent João

CLI Python sobre a IPED Web API com triagem determinística, banco local de entidades, padrões investigativos e
complementação por LLM sob revisão humana.

### OpenClaw + Linear + Slack Workflow

Orquestração multiagente para ligar contexto, execução, rastreio e comunicação operacional.

## Prioridades Técnicas

- IA aplicada com rastreabilidade e revisão humana
- Tooling que reduz atrito operacional de verdade
- Workflows baseados em repositório, diff e histórico
- Sistemas investigativos e forenses com foco prático
- Estruturas de conhecimento reutilizáveis para trabalho contínuo

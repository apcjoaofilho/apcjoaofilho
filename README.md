## Sobre

Atuo na interseção entre engenharia de software, IA aplicada e automação de workflows. Tenho interesse especial em sistemas úteis e auditáveis: integração com modelos, publicação baseada em Git, tooling para agentes, análise documental, OCR, fluxos investigativos e aplicações de IA para o setor público.

Meu foco não é demonstrar tecnologia por si só, mas construir fluxos que funcionem sob restrições reais: contexto incompleto, necessidade de validação, rastreabilidade e operação contínua.

## Projetos e Frentes de Trabalho

### [JBA Desenvolvimento](https://www.joaobarbosa.ia.br/)

Plataforma e operação de conteúdo para consultoria e treinamentos em IA voltados ao setor público, com publicação baseada em repositório, Markdown, GitHub e deploy automático na Vercel.

### PiauEye

Arquitetura de plataforma de análise forense digital com IA híbrida, combinando processamento local, automação de ingestão, extração de entidades e organização de evidências para contexto policial brasileiro.

### Agent João

CLI Python para operar sobre a IPED Web API com triagem determinística, banco local de entidades, padrões investigativos e complementação por LLM sob revisão humana.

### [Digital Garden / joao-filho](https://github.com/apcjoaofilho/joao-filho)

Fluxo pessoal de publicação versionada com notas em Markdown, GitHub e Vercel, usado como base pública de conhecimento, autoridade e reaproveitamento de conteúdo.

### OpenClaw + Linear + Slack Workflow

Orquestração multiagente para fechar o ciclo entre contexto, execução, rastreio e comunicação operacional.
